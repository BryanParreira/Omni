### Step-by-Step Setup

1. **Unzip the File**: Download the release and unzip the folder.
2. **Move the App**: Open the unzipped folder and drag `Omni.app` into your Applications folder.
3. **Authorize the App**: Open the Terminal app and run the following command. This fixes the "App is damaged" error caused by macOS quarantine:

```bash
xattr -cr /Applications/Omni.app
```

4. **You are good to go!** Launch Omni from your Applications folder.
